h1. EEProgressHUD

EEProgressHUD is an easy-to-use, clean and lightweight progress HUD for iOS.

EEProgressHUD features:

* singleton
* call @+ (void)showWithMessage:showStyle:progressViewStyle:@ when you want to show HUD
* call @+ (void)hideWithMessage:hideStyle:resultViewStyle:@ when you want to hide HUD
