h1. EEProgressHUD

EEProgressHUD is an easy-to-use, clean and lightweight progress HUD for iOS.

!https://lh3.googleusercontent.com/-CACjjz0IPv8/TuC6DepBWsI/AAAAAAAAAEo/FuDBSejzp0k/s912/screenshots2.png!

EEProgressHUD features:

* singleton class
* call @+ (void)showWithMessage:showStyle:progressViewStyle:@ when you want to show HUD
* call @+ (void)hideWithMessage:hideStyle:resultViewStyle:@ when you want to hide HUD
* showStyle: How to show HUD
* progressViewStyle: How to display in progress
* hideStyle: How to hide HUD
* resultViewStyle: last image type

h2. Installation

* Drag the @EEProgressHUD.h@ & @EEProgressHUD.m@ into your project.
* Add the *QuartzCore* framework to your project
